# Matugen Integration
Basic integation of Matugen using swww for gtk3,gtk4,qt5ct,qt6ct,kde

# Setup
Copy the contents of the repo to `~/.config/matugen/`  
Or use this command  
```sh
git clone https://github.com/alexankitty/matugen-integration ~/.config/matugen
```
